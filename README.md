# Installation

# Vue Task List
Completed Vue Task List that is hosted on https://www.alexgellert.com/vuelist.html

# Functions
```
1. Choose Level of Importance
2. Add to Task List
3. Re-order Task List
4. Complete Task
5. Delete Task
6. Save Current Task List
```
